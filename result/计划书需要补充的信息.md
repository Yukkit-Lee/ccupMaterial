# 智检云盾商业计划书需要补充的信息

以下内容已在《智检云盾商业计划书（正式提交版）》中以红色“此处填写……”占位符标出。请先准备真实、可核验的材料，再替换占位符；没有发生或暂无材料的事项，应如实写“暂无/尚未开展”，不要编造。

## 一、参赛与主体材料

1. 申报单位名称、项目联系人、联系电话、电子邮箱及提交日期。
2. 永清县臻青奇百货店的营业执照扫描件、实际经营范围，以及该主体承载本项目的说明；如后续设立新公司，补充新主体名称、统一社会信用代码、股权关系和承接安排。
3. 团队成员完整名单、学校/专业/年级或岗位、个人联系方式，以及每位成员真实承担的工作量。
4. 指导教师或顾问的姓名、单位、职务、实际指导内容和本人授权；若赛事要求匿名，应按要求另行脱敏。

## 二、技术与知识产权材料

1. 改进 DPSGD 算法的技术说明：改进点、适用条件、与基线方法的对比口径、代码版本号和实验记录位置。
2. 可复现实验材料：数据集来源或授权文件、样本数量、训练/验证/测试划分、参与节点数、硬件与软件环境、评价指标、实验日期和结果。
3. 原型系统材料：产品截图、部署架构图、演示视频链接、代码仓库地址、软件著作权/专利申请状态（仅填已真实取得或已真实提交的内容）。
4. 第三方组件与开源依赖清单、许可证，以及自主代码与第三方代码的边界说明。

## 三、市场与客户验证材料

1. 用户访谈记录：受访企业/岗位、访谈日期、痛点、现有流程、预算区间、是否同意引用；未获授权时使用匿名表述。
2. 潜在客户或合作方材料：真实名称仅在获得授权后使用；否则填写行业、区域、任务类型和接触进度。意向函、试点协议、数据授权或保密协议如有，应附复印件或编号。
3. 目标市场数据的来源、发布时间、统计口径及链接；不能仅写“市场巨大”。
4. POC 的验收指标：缺陷类别、准确率/召回率/误报率、时延、通信量、隐私预算、人工复检规则和验收责任人。

## 四、商业与财务材料

1. 标准报价、直接成本、项目工时、差旅、算力、接口适配和税费的实际测算依据；如报价尚未形成，保留“拟定”并标注调研日期。
2. 三年预测的客户漏斗依据：线索数、POC 数、部署数、订阅节点数、转化率、续费率、回款周期和假设说明。
3. 现有资金来源、实际可投入金额、已发生支出、拟融资金额/方式/用途；没有融资时如实填写“暂无外部融资”。
4. 记账、开票、纳税和数据处理责任的安排；涉及个人信息或重要数据时，补充合规评估与合同条款。

## 五、提交前核对

1. 将正式文档中的所有红色占位符逐项替换或明确删除。
2. 复核统一社会信用代码、成立日期、资金数额和经营状态是否与营业执照一致。
3. 检查学校名称、标志、指导教师姓名等是否符合赛事匿名/脱敏要求。
4. 仅保留能提供证据支持的客户、技术效果、知识产权、收入、融资和合作表述。
