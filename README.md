# 智检云盾｜挑战杯东北振兴产业升级专项赛材料库

项目名称：**智检云盾——面向装备制造的多工厂数据安全协同质检平台**  
参赛组别：东北振兴产业升级专项赛·数智赋能产业

本仓库用于团队协同沉淀赛事资料、技术参考、写作依据和阶段性成果。核心场景是：多家装备制造企业不共享原始质检图像与工艺数据，也能协同提升缺陷识别能力。

## 先这样使用本仓库

开始任何写作、调研或开发前，请把下列材料提供给大模型阅读，并要求它先复述理解、再开展任务：

1. `通知说明.txt`、`赛事介绍.txt`、`提交所需材料.txt`：赛事赛道、申报要求和材料规范。
2. `项目库.txt`：往届项目库入口，用于参考项目名称和表达方式。
3. `reference/`：技术参考文献。外部论文、开源组件和公开资料仅用于学习与对照，**不得写成团队自有论文、专利或既有成果**。
4. `写作指导png/`：计划书写法与财务测算的案例。重点是收入必须由客户数、单价、续费和交付能力逐项推导，不能用市场份额倒推或编造数字。
5. `商业计划书与汇报模板/`：官方模板与格式要求；不要直接修改模板文件，应在其框架和格式要求下输出新文件。
6. `result/`：大模型输出、计划书草稿和对话记录均保存在此目录。`result/conversation.md` 按对话以 `---` 分隔追加。

## 协作规则

- 不覆盖原始通知、模板、参考论文和指导图片；新成果统一保存到 `result/`。
- 客户、合同、数据授权、营收、融资、专利、团队履历等事实，必须有团队提供的证据或明确标注为“预测/待核验”。
- 所有引用的论文、政策、市场数据和企业信息应保留来源；不得虚构合作关系或技术成果归属。
- 提交版本须按赛事要求隐去学校名称、标志、指导教师姓名等推报单位信息。
- 每次实质修改后，检查变更、提交清晰的 Git 提交说明并推送到远程仓库。
